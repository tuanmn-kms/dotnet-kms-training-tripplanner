# ?? KMS Training - Trip Planner

Full-stack trip planning app with .NET 10 + React 19

## Features
- Trip/Destination/Activity/Budget Management
- JWT Authentication
- Modern React UI
- Docker Support
- 49 Unit Tests

## Quick Start
Backend: `cd KMSTraining.API && dotnet run`
Frontend: `cd KMSTraining.ReactUI && npm run dev`

## Tech Stack
.NET 10 | React 19 | TypeScript | Tailwind CSS | SQL Server
